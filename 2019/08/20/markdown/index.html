Markdown 语法说明

一、标题

  • 使用 # 控制
  • 几个 # 代表几级标题
  • #与文字之间有空格

二、强调

1. 加粗

在要加粗的字段两侧添加 **
这里加粗强调

2. 斜体

在要加斜体的字段两侧添加 *
这里需要加斜体强调

3. 加粗斜体

在字段两侧添加*
这里需要 *加粗并斜体***

4.删除

在要删除的字段两侧添加 ~
~这句话需要删除~~

三、列表

1、无序列表

  • 项目1
  • 项目2
    • 项目2.1
    • 项目2.2

2、有序列表

  1. 项目1
  2. 项目2
  3. 项目3
    1. 项目3.1
    2. 项目3.2
